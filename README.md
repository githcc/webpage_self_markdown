## csdn 离线化方式
1. 使用选择edge，选中文章，使用沉浸式阅读器打开
2. 设置好Typora插入图片时存储图片的目录
3. 复制沉浸式阅读器的内容到一个md文件即可

## 飞书 离线化方式
方式一：
1. ctrl+P调用打印，另存为pdf
2. 去除页眉和顶脚

方式二：推荐
1. 使用ctrl+A两次选择内容
2. 正常粘贴即可

## notion 离线化
1. 将网页ctrl+S保存到本地
2. 将网页上传至服务器
3. 重新打开网页
4. 使用ctrl+A选择内容
5. 正常粘贴即可
